## 📝 소개
> Biconnect에서 실사용을 위해 제작한 웹페이지입니다.

프리팅 입점 병원 정보를 표시하고 필터링하는 웹 애플리케이션으로, 기능은 아래와 같습니다.

- 병원 정보 표시 (이름, 주소, 태그, 대표원장, 운영시간)
- 병원 상세 정보 아코디언 기능 (병원 소개, AI 인사이트)
- 검색 기능 (병원명, 주소, 의사, 태그)
- 태그 기반 필터링
